A collection of lightweight, low-refresh web games built specifically for the **Kobo Clara Colour** e-reader browser.

## 🐍 Games Included

* **Turn-Based Snake:** Controlled via an on-screen D-Pad. Built with HTML5 Canvas using lightweight, high-contrast colors and zero auto-timers to suit e-ink display refresh rates.

---

## 🛠️ E-Ink Compatibility Rules Used

* **No continuous animations:** All interactions are step- or turn-based.
* **Basic HTML/CSS:** Built with standard layout elements (`<table>`, raw CSS) for maximum compatibility with stripped-down e-reader WebKit browsers.
* **Large Touch Targets:** Buttons are designed for easy tapping without accidental misclicks.
